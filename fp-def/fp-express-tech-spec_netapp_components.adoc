ネットアップストレージコントローラは、ブートとアプリケーションデータストレージの両方を実現する FlexPod Express アーキテクチャのストレージ基盤を提供します。このセクションでは、 FlexPod Express アーキテクチャのさまざまなネットアップオプションについて説明します。



== ネットアップストレージコントローラのオプション



== NetApp FAS

AFF C190、AFF A220、またはFAS2750シリーズの冗長コントローラは、FlexPod Expressアーキテクチャに必要です。コントローラはONTAPソフトウェアを実行します。ストレージコントローラを注文する際、希望するソフトウェアバージョンをコントローラにプリロードすることができます。ONTAPの場合、クラスタはクラスタ インターコネクトスイッチのペアを使用してデプロイするか、スイッチレスクラスタ構成でデプロイすることができます。

== ネットアップのディスクシェルフとディスク



== NetApp FAS

ストレージコントローラには、最低1つのNetApp ディスク シェルフが必要です。選択したNetApp シェルフタイプによって、そのシェルフ内で使用可能なドライブの種類が決まります。

FAS2700およびFAS2600シリーズのコントローラは、デュアルストレージコントローラとディスクを同一シャーシ内に搭載した構成で提供されます。この構成はSATAまたはSASドライブに対応しているため、性能や容量の要件でスピンドル数を増やす必要がない限り、追加の外付けディスク シェルフは不要です。

[NOTE]
====
すべてのディスク シェルフの部品番号は、AC電源ユニット2台を備えた空のシェルフのものです。その他の部品番号については、営業担当者にお問い合わせください。

ディスクドライブの部品番号は、購入予定のディスクのサイズとフォームファクタによって異なります。その他の部品番号については、営業担当者にお問い合わせください。

====

=== Eシリーズ ストレージ

シャーシ内にドライブを搭載していないストレージコントローラには、最低1つのNetApp ディスク シェルフが必要です。選択したNetAppシェルフタイプによって、そのシェルフ内で使用可能なドライブの種類が決まります。
