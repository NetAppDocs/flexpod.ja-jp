= FlexPod ルール
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod の設計は、多様なハードウェア コンポーネントとソフトウェア バージョンを包含する柔軟なインフラストラクチャ アーキテクチャをサポートします。

ルールセットをガイドとして使用して、有効なFlexPod構成を構築またはアセンブルしてください。この文書に記載されている数値とルールは、FlexPodの最低限の要件を表しており、さまざまな環境やユースケースに応じて、付属の製品ファミリー内のサポート対象製品を使用することで拡張できます。
