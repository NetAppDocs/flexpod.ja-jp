= シスコのコンポーネント

:allow-uri-read: 
:nofooter:

[role="lead"]
シスコは、 FlexPod Express の設計およびアーキテクチャに大きく貢献しており、解決策のコンピューティングおよびネットワーキングレイヤに貢献しています。ここでは、 FlexPod Express で使用できる Cisco UCS および Cisco Nexus コンポーネントについて説明します。



== Cisco UCS X シリーズダイレクト

Cisco UCS X-Series Directは、Cisco UCS Fabric Interconnect 9108 100GのペアとCisco UCS X9508シャーシおよびCisco UCS X-Seriesコンピュートノードを統合し、UCS X-Series Directソリューションを提供する新しいアーキテクチャです。

Cisco UCS X シリーズの機能と特徴の概要については、link:https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-aag.html["Cisco UCS X シリーズ ダイレクトの概要"^]を参照してください。

Cisco UCS X-Series Directは、以下のコンポーネントで構成されています。

* Cisco UCS 9108 100G ファブリック インターコネクト
* Cisco UCS X9508シャーシ
* Cisco UCS Xシリーズコンピュートノード
* オプションコンポーネント：
+
** PCIe ノードと X-Fabric モジュール
** 2番目の Cisco UCS X9508 シャーシ（X シリーズコンピューティングノードおよび Cisco UCS 9108-100G インテリジェントファブリックモジュール搭載）と最大 4 台の C シリーズラックサーバー





NOTE: UCS X-Series Direct の完全な技術仕様については、こちらのリンクを参照してください： https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-ds.html["Cisco UCS X シリーズダイレクトデータシート"]。
