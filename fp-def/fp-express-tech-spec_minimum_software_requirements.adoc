== FlexPod Express with NetApp AFF または FAS のソフトウェア要件

NetApp FASを搭載したFlexPod Expressのソフトウェア要件は次のとおりです：

* ONTAP 9.1 以降
* Cisco NX-OSバージョン7.0(3)I6(1)以降
* Cisco UCS管理構成では、Cisco UCS Manager UCS 4.0(1b)


すべてのソフトウェアはリストに記載され、 http://mysupport.netapp.com/matrix["NetApp IMT"^]でサポートされている必要があります。特定のソフトウェア機能によっては、以前のアーキテクチャで記載されていた最小バージョンよりも新しいバージョンのコードが必要になる場合があります。
