== NetApp AFF/ ASA/ FASを搭載したFlexPod Expressのソフトウェア要件

NetApp AFF/ ASA/ FASを搭載したFlexPod Expressのソフトウェア要件は次のとおりです：

* ONTAP 9.16 以降
* Cisco NX-OSバージョン10.4（6）M以降



NOTE: このセクションに記載されているすべてのソフトウェアバージョンは、ドキュメントの後続のセクションに記載されているハードウェアコンポーネント（NetAppおよびCiscoコンポーネント）の最小要件です。

すべてのソフトウェアがにリストされ、サポートされている必要があります http://support.netapp.com/matrix/["NetApp IMT"^]。一部のソフトウェア機能では、以前のアーキテクチャに示されている最小要件よりも新しいバージョンのコードが必要になる場合があります。
