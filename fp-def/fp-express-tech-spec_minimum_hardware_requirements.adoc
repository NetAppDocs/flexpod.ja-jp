== FlexPod Express（NetApp AFF/ ASA/ FAS搭載）

基盤となるストレージにNetApp AFF、ASA、またはFASコントローラを使用するFlexPod Expressソリューションのハードウェア要件には、本セクションで説明する構成が含まれます。

FlexPod Express の最小ハードウェア要件は次のとおりです：

* 冗長構成の 2 台の 10Gbps 標準イーサネット スイッチ（Cisco Nexus 9200 および 9300 モデル）
* Cisco UCS C シリーズスタンドアロンラックサーバー（CIMC ベースのスタンドアロンラックサーバー構成または Intersight 管理構成（IMM））、または Cisco UCS X シリーズダイレクト（Cisco IMM 構成）
* ハイアベイラビリティ（HA）ペア構成で2ノードクラスタとして導入される、エントリーレベルのNetApp AFF（A20、A30、またはC30）、NetApp ASA（A20、A30、またはC30）、またはNetApp FAS（2750または2820）シリーズコントローラ2台
