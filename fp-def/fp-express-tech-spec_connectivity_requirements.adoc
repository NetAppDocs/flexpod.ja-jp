== NetApp AFF/ ASA/ FASを搭載したFlexPod Expressの接続要件

FlexPod Express（NetApp AFF/ ASA/ FAS 搭載）の接続要件は次のとおりです：

* NetAppストレージコントローラは、Cisco Nexusスイッチに直接接続する必要があります。
* UCS Xシリーズのダイレクト展開の場合、NetAppストレージコントローラは、組み込みのファブリック インターコネクトに直接接続できます。
* FlexPod のコアコンポーネント間には、追加の機器をインラインで配置できません。
* 仮想ポートチャネル（vPC）は、Cisco Nexus 9000シリーズスイッチをNetAppストレージコントローラに接続するために必要です。
* 必須ではありませんが、環境全体でジャンボフレームのサポートを有効にすることを推奨します。
