== NetApp AFF/ ASA/ FASを搭載したFlexPod Expressの接続要件

NetApp AFF/ ASA/ FASを搭載したFlexPod Expressの接続要件は、一般的に次のとおりです：

* NetAppストレージコントローラは、Cisco Nexusスイッチに直接接続する必要があります。
* Cisco UCS X-Series Direct 展開では、NetApp ストレージコントローラを組み込みのファブリックインターコネクトに直接接続できます。
* FlexPod のコアコンポーネント間には、追加の機器をインラインで配置できません。
* 仮想ポートチャネル（vPC）は、Cisco Nexus 9000シリーズスイッチをNetAppストレージコントローラに接続するために必要です。
* 必須ではありませんが、環境全体でジャンボフレームのサポートを有効にすることを推奨します。


各トポロジーに固有の接続要件については、link:https://docs.netapp.com/us-en/flexpod/fp-def/fp-express-tech-spec_flexpod_express_topologies.html["FlexPod Express トポロジー"^]セクションを参照してください。
