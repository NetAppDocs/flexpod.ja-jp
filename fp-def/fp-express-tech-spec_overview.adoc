= FlexPod Express 技術仕様
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod Expressは、Cisco Unified Computing System（Cisco UCS）とCisco Nexusファミリーのスイッチを基盤とし、ストレージ レイヤはNetApp AFF、ASA、またはFASエントリーレベルのストレージシステムを使用して構築された、事前設計されたベストプラクティスアーキテクチャです。FlexPod Expressは、さまざまな仮想化ハイパーバイザーやベアメタルオペレーティングシステム（OS）、エンタープライズワークロードを実行するのに適したプラットフォームです。

FlexPod Expressは、ベースライン構成を提供するだけでなく、さまざまなユースケースや要件に対応できるよう、サイズ設定や最適化を柔軟に行えるという利点もあります。
