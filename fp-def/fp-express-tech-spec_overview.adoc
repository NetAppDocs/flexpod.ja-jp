= FlexPod Express 技術仕様
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod Express は、Cisco Unified Computing System （Cisco UCS）と Cisco Nexus ファミリーのスイッチをベースに構築された、ベストプラクティスに基づく事前設計のアーキテクチャです。ストレージ レイヤは、NetApp AFF、ASA、または FAS エントリーレベルのストレージ システムを使用して構築されています。FlexPod Express は、中小規模の環境、リモートオフィスおよびブランチオフィス （ROBO） の展開、エッジロケーションにおいて、さまざまな仮想化ハイパーバイザー、ベアメタルオペレーティングシステム （OS）、およびエンタープライズワークロードを実行するのに適したプラットフォームです。

FlexPod Expressは、ベースライン構成を提供するだけでなく、さまざまなユースケースや要件に合わせて拡張および最適化できる柔軟性も備えています。
