= TR-4293: FlexPod Express 技術仕様概要
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


カーシック・ラダクリシュナン、アルビンド・ラマクリシュナン、リンジー・ストリート、サビタ・クマリ、NetApp

[role="lead"]
FlexPod Expressは、Cisco統合コンピューティングシステム（Cisco UCS）とCisco Nexusファミリーのスイッチをベースに構築された、ベストプラクティスに基づく事前設計のアーキテクチャであり、ストレージ レイヤはNetApp FASまたはNetApp Eシリーズ ストレージを使用して構築されています。FlexPod Expressは、さまざまな仮想化ハイパーバイザーやベアメタルオペレーティングシステム（OS）、エンタープライズワークロードを実行するのに適したプラットフォームです。

FlexPod Expressは、ベースライン構成を提供するだけでなく、さまざまなユースケースや要件に合わせて拡張および最適化できる柔軟性も備えています。本書では、使用するストレージ システムに基づいて、FlexPod Express with NetApp FASおよびFlexPod Express with Eシリーズの構成に分類して説明します。
