== NetApp ONTAP

NetApp ONTAP ソフトウェアは、AFF、ASA、FAS ストレージ システムで動作するオペレーティング システムです。ONTAP は、拡張性の高いストレージ アーキテクチャを提供し、無停止運用、無停止アップグレード、俊敏なデータ インフラストラクチャを実現します。

ONTAP の詳細については、を参照してください http://www.netapp.com/us/products/platform-os/ontap/index.aspx["ONTAP の製品ページ"^]。
