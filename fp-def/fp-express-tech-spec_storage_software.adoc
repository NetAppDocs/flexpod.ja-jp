== NetApp ONTAP

NetApp ONTAP ソフトウェアは、AFF、ASA、および FAS ストレージ システム上で動作するオペレーティング システムです。ONTAP は、拡張性の高いストレージ アーキテクチャを提供し、無停止運用、無停止アップグレード、および俊敏なデータ インフラストラクチャを実現します。ONTAP One ライセンス モデルにより、すべてのエンタープライズ データ サービスが単一のパッケージに含まれ、導入と管理が簡素化されるとともに、高度な機能への完全なアクセスが提供されます。ONTAP は、レプリケーションおよびディザスタ リカバリのための SnapMirror、アプリケーション整合性のあるバックアップとリストアのための SnapCenter などのテクノロジーを通じて、堅牢なデータ保護を提供します。また、自律型ランサムウェア対策や暗号化などの組み込みのセキュリティ機能を備え、現代の脅威からデータを保護します。さらに、ONTAP は SnapMirror Cloud や S3 ベースのレプリケーションなどの機能により、シームレスなハイブリッド クラウド統合を実現し、一貫性と制御を維持しながらオンプレミスとクラウド環境間のデータ移動を可能にします。

ONTAP の詳細については、 https://www.netapp.com/ontap-data-management-software/["ONTAP の製品ページ"^]を参照してください。
