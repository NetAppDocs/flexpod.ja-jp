= 概要
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
このテクニカルレポートでは、 FlexPod 上の VMware vSphere 7 での NVMe over Fibre Channel （ FC-NVMe ）パフォーマンスの概要を説明します。FlexPod は、 FC-NVMe を導入してアプリケーションへのハイパフォーマンスなストレージアクセスを実現するための最適な共有インフラ基盤です。
